
# netflix-gpt
Welcome to Netflix GPT, a Netflix clone that offers a familiar interface to browse movies. Built with HTML, CSS, Tailwind CSS, JavaScript, React.js, and Redux, it integrates with TMDB for movie details and Firebase for authentication. Experience modern UI/UX and efficient state management in this project.
